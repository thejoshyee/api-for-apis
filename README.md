# Scrimba - API Challenge

## About
This mini challenge was for Scrimba's Front End Developer Career Path. The target focus was using Fetch to gather API data and display it on a page using CSS grid and CSS flexbox. 

## Learning:
- CSS Grid</br>
- CSS FlexBox</br>
- Fetch</br>
